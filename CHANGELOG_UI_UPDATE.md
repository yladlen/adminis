#### 2. ✅ Кнопка "Сдан" теперь заполняет всю ячейку
**Файл:** `laptops/index.php`

**Изменения:**
- Кнопка "✅ Сдан" теперь занимает всю ячейку таблицы
- Убраны отступы (padding: 0 на ячейке)
- Кнопка имеет класс `btn-success` с полной шириной и высотой
- Улучшенная визуальная привлекательность
- Центрирование текста внутри кнопки

**Код:**
```php
<td class="p-0" onclick="event.stopPropagation();">
    <?php if ($row['status'] === 'взят'): ?>
        <a href="../mark_returned.php?id=<?= $row['id'] ?>"
        onclick="return confirm('Подтвердить возврат ноутбука?')"
        class="btn btn-success w-100 h-100 d-flex align-items-center justify-content-center"
        style="border-radius: 0; margin: 0; padding: 8px;">
        ✅ Сдан
        </a>
    <?php else: ?>
        <span class="text-muted d-block text-center">—</span>
    <?php endif ?>
</td>
```

### Рекомендации по дальнейшему использованию

1. **Для добавления новых страниц:**
   - Подключайте `includes/top_navbar.php` после `includes/navbar.php`
   - Используйте стандартную структуру `.layout-wrapper` > `.sidebar` + `.content`
   - Подключайте `includes/unified_styles.css` для консистентности

2. **Для кнопок, заполняющих ячейку:**
   - Используйте класс `p-0` на `<td>`
   - Добавьте классы `w-100 h-100 d-flex align-items-center justify-content-center` на кнопку
   - Установите `border-radius: 0` для кнопки

3. **Для обновления названий разделов в navbar:**
   - Редактируйте массив `$page_titles` в файле `includes/top_navbar.php`
