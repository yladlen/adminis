# Adminis — Учёт и визуализация сети

🖥️ Веб-приложение для системных администраторов: инвентаризация оборудования,
карта сети, журнал выдач, мониторинг серверов и документация.

## 📦 Возможности

- **Устройства** — учёт ПК, серверов, принтеров, свитчей, роутеров, ИБП по кабинетам
- **Карта сети** — интерактивный force-directed граф с фильтрацией, мультивыбором и экспортом PDF
- **История и проблемы** — журнал изменений устройств, открытые/закрытые неисправности
- **Сотрудники** — база сотрудников с привязкой к кабинетам и должностям
- **Журнал выдач** — учёт выданных ноутбуков с датами и статусами
- **Серверные обходы** — фиксация визитов в серверные, чек-листы, проблемы
- **Документация** — иерархическое дерево с drag & drop, inline-переименование, поиск
- **Главная страница** — сводка: открытые проблемы, последние выдачи, статусы
- **Миграции** — система версионных миграций БД через веб-интерфейс
- **Установка** — пошаговый мастер с проверкой окружения

## 📂 Структура проекта
```
adminis/
├── includes/        # DB, конфиг, navbar, стили
├── setup/           # Мастер установки и миграции
├── rooms/           # Кабинеты и устройства
├── computers/       # Раздел ПК
├── notebooks/       # Раздел ноутбуков
├── servers/         # Раздел серверов
├── printers/        # Раздел принтеров
├── ups/             # Раздел ИБП
├── employees/       # Сотрудники
├── journal/         # Журнал выдач
├── map/             # Карта сети
├── docs/            # Документация
├── assets/icons/    # Иконки устройств
└── index.php        # Главная страница
```

## 🚀 Установка

1. Разместите проект в `/var/www/html/adminis`
2. Создайте базу данных MySQL / MariaDB
3. Перейдите по адресу `http://your-domain/adminis/setup/`
4. Пройдите мастер установки
5. После установки откройте `setup/migrate.php` для применения миграций

## ⚙️ Требования

- PHP 7.4+
- Расширения: `pdo`, `pdo_mysql`, `mbstring`, `openssl`, `json`
- MariaDB / MySQL

## 🗺️ Карта сети

- Автоматическая раскладка (force-directed граф)
- Фильтрация по кабинету и типу устройства
- Слайдер расстояния между узлами
- Мультивыбор рамкой, перетаскивание групп
- Панорама колесом мыши / средней кнопкой, зум Ctrl+колесо
- Экспорт в PDF

## 🔐 Авторизация

- Логин и пароль задаются при установке
- Хранятся в `includes/config.php`
